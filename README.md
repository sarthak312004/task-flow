# Task~Flow

A simple task management application with an integrated timer to track time spent on each task.

## Features

- ⏱️ **Timer** - Track hours, minutes, and seconds for your tasks
- ➕ **Add Tasks** - Create new tasks with a simple text input
- ✅ **Mark Complete** - Check off completed tasks
- 📊 **Progress Tracking** - Automatically track time spent on selected tasks
- 🗑️ **Delete Tasks** - Remove tasks you no longer need
- 🎨 **Glowing Effect** - Visual feedback when timer is running

## Getting Started

Simply open `index.html` in your browser to start using Task~Flow.

## How to Use

1. Enter a task name in the input field and click **ADD**
2. Click the **play button** to start the timer
3. Select a task to track time for that specific task
4. Use the **checkbox** to mark tasks as complete
5. Click the **trash icon** to delete tasks
6. Click **reset** (when timer is running) to clear the counter

## Files

- `index.html` - Main application interface
- `script.js` - Core functionality and timer logic
- `style.css` - Styling and layout
- `res/` - Image assets for icons and buttons
